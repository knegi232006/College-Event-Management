College Event Management System

A simple console-based College Event Management System developed in C Programming Language.
This project helps colleges manage events and student registrations digitally instead of using manual paper-based systems.
  
   Features
Admin Module
Add new events
View all events
View student registrations
Store event data permanently using files
Student Module
View available events
Register for events
Simple menu-driven interface

   How to Run
Install GCC Compiler
Open terminal/command prompt
Compile the program:
gcc main.c -o event
Run the program:
./event
  Main Menu
1. Add Event (Admin)
2. View All Events
3. Register for Event (Student)
4. View Registrations
5. Exit
    Data Storage
The project uses binary files for permanent storage:
events.dat → Stores event details
students.dat → Stores student registration data
